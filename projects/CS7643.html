CS7643 – Deep Learning

A collection of projects exploring foundational and advanced deep learning techniques through practical, research-inspired work.

CNN for CIFAR-10 Image Classification

Using PyTorch, I built a convolutional neural network from the ground up to classify images from the CIFAR-10 dataset. I experimented with class imbalance by modifying the dataset distribution and applied a range of strategies—data augmentation, re-weighting, and sampling—to improve model robustness.

Through careful tuning and experimentation, I observed how different techniques impacted precision and recall on underrepresented classes. This project not only sharpened my PyTorch fluency but also gave me practical insight into the importance of fairness and balance in model performance.

Machine Translation with RNN, LSTM, Seq2Seq, and Transformer Models

This multi-phase project focused on building and comparing several deep learning architectures for language translation. I began by implementing RNN and LSTM cells from scratch, including parameter initialization and forward logic—gaining firsthand appreciation for how sequential data flows through a network.

Next, I developed a Seq2Seq model with cosine similarity-based attention, followed by a Transformer encoder with multi-head self-attention, residual connections, and layer normalization. I trained all models and compared perplexity and output quality, analyzing how each architecture handled long-term dependencies and context representation.

This project pushed me to think deeply about architecture design choices, and how small differences (like attention vs. recurrence) can yield very different learning behaviors.

Efficient LLM Fine-Tuning via Knowledge Distillation

In our final project, my team and I tackled large language model fine-tuning using knowledge distillation. We used a teacher-student framework with temperature scaling, soft-label distillation, and KL-divergence loss blending to fine-tune smaller student models efficiently.

We trained models on MNLI and CoLA (with validation via HANS and CoLA-OOD), comparing performance across Vanilla Fine-Tuning, PBFT, and In-Context Learning. Our experiments used models ranging from 125M to 6.7B parameters, and we evaluated both accuracy and memory use.

After early limitations on Google Cloud and Colab, we transitioned to PACE with 2x H100 GPUs, using Docker to ensure reproducibility. The larger models outperformed in-context methods, but training stability and parallelization remained tricky, particularly with DeepSpeed.

What stood out most was the value of systematic experimentation—understanding when a smaller model, well-tuned, can match or exceed the performance of a much larger one.

Machine Learning for Trading

Course: CS7646 – Fall 2023 | Georgia Institute of Technology

Overview

This project was a deep dive into designing an algorithmic trading system that leverages machine learning to make informed decisions. The primary goal was to build a fully automated pipeline capable of identifying trade opportunities using both technical indicators and ensemble-based learning models. The project was conducted as part of the Machine Learning for Trading course and focused heavily on reproducibility, performance analysis, and robustness to real-world trading constraints.

What I Built

I developed a Python-based automated trading strategy that combines traditional technical analysis with modern machine learning techniques. Indicators such as EMA, SMA, Bollinger Bands, and CCI were used to generate meaningful features for a bag learner composed of random tree learners. I also implemented a full market simulator that accounted for key market dynamics—transaction costs, commission fees, and market impact. The strategy was optimized and tuned extensively to improve predictive performance and generalizability.

What I Learned

This project helped me understand the nuances of financial time series data and how difficult it can be to make predictions in a noisy market. I learned the importance of balancing model complexity with generalizability and how critical it is to validate results across different market conditions. Most importantly, I gained hands-on experience with backtesting, feature engineering, and measuring trading performance using cumulative returns and Sharpe ratios.

Challenges

One of the biggest challenges was ensuring the strategy performed well not just in-sample, but also out-of-sample. While the model achieved an impressive 102% cumulative return in-sample, it delivered a more modest but realistic 10% cumulative return out-of-sample. Tuning the learner to handle transaction costs and avoiding overfitting were key technical hurdles. Building a market simulator that realistically modeled execution costs also required careful design.
